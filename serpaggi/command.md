# Shell command cheat sheet :

Unzip file
```
$> gunzip data.gz
```

Type data file
```
$> file data
```

Info sur un document
```
ls -l data
```

Nb ligne du doc
```
$> wc -l data
```

Obtenir le contenu d'une variable
```
$> echo X
```

Obtenir le contenu d'un file
```
$> cat <file>
```

Dowload file
```
$> curl -O <URL_TO_FILE>
```

Find things in file
```
$> grep <file>
```
